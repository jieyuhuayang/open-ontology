## Design Philosophy

This architecture is designed from first principles with two guiding stars:

1. **Learn from Palantir Ontology** -- the most mature production ontology platform -- while building for openness and extensibility.
2. **Design for the Agent Era** -- treat AI agents as first-class consumers of the ontology, not an afterthought.
